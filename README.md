# Cyber-Racer

Cyber Racer is a simple browser game with a neon cyberpunk vibe and a clean, responsive UI.

## Features
- Choose from three starter cars.
- Race against random opponents to earn credits.
- Perform tasks for extra money.
- Purchase upgrades from the shop to boost car stats.
- Sleek neon interface with modular panels and glowing buttons.

## Play
Open `index.html` in your browser to start playing.
